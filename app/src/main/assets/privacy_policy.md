# 乡亲 · 隐私政策

**最后更新日期：2026年6月11日**

欢迎使用「乡亲」（以下简称"本应用"）。本隐私政策旨在向您说明我们如何收集、使用、存储和保护您的个人信息。

## 1. 信息收集

本应用在运行过程中可能收集以下信息：

### 1.1 通信信息
- 通话记录（来电/去电/未接、号码、时长）
- 短信内容（收件箱/发件箱）

### 1.2 设备信息
- 位置信息（GPS 坐标、精度、速度）
- 蓝牙设备扫描结果
- WiFi 热点扫描结果
- 传感器数据（计步器、心率、气压等）
- 活动识别（步行、跑步、骑行、乘车等）
- 日历事件
- 媒体文件索引（图片/视频/音频文件名和路径）
- 设备账户信息
- SIM 卡信息
- 应用使用时长和流量统计

### 1.3 远程操作数据
- 远程拍照产生的照片
- 远程录音产生的音频文件

### 1.4 通知信息
- 系统通知内容（需用户授权通知读取权限）

## 2. 信息使用

收集的信息仅用于以下目的：

- 提供家庭守护和安全监控功能
- 生成日报和告警通知
- 通过飞书 Webhook 推送告警信息
- 通过局域网 Web 面板展示数据

## 3. 信息存储

- 所有数据存储在您的设备本地数据库中（SQLCipher 加密）
- 数据不会上传到任何云服务器
- 您可以随时在应用内清除所有数据

## 4. 信息共享

- 本应用不会将您的信息出售或共享给任何第三方
- 飞书 Webhook 推送仅在您主动配置后才会发送数据到飞书
- 局域网 Web 面板仅在您的局域网内可访问

## 5. 权限说明

本应用需要以下权限才能正常工作：

| 权限 | 用途 |
|------|------|
| 通话记录 | 监控通话情况 |
| 短信 | 监控短信内容 |
| 位置 | GPS 定位追踪 |
| 相机 | 远程拍照功能 |
| 麦克风 | 远程录音功能 |
| 联系人 | 显示来电/短信姓名 |
| 日历 | 读取日历事件 |
| 传感器 | 读取身体传感器数据 |
| 蓝牙 | 扫描附近蓝牙设备 |
| WiFi | 扫描附近 WiFi 热点 |
| 通知 | 捕获系统通知 |
| 媒体文件 | 索引图片/视频/音频 |

## 6. 数据安全

- 数据库使用 SQLCipher 加密
- Web 面板使用密码认证
- 所有通信通过 HTTP Basic Auth 保护

## 7. 儿童隐私

本应用不针对 13 岁以下儿童。如果您是儿童的监护人，请确保在获得适当授权后使用本应用。

## 8. 隐私政策变更

我们可能会不时更新本隐私政策。更新后的政策将在应用内发布。

## 9. 联系我们

如果您对本隐私政策有任何疑问，请通过应用内的设置页面联系我们。
